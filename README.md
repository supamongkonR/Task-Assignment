# Assignment
Design system briefly by looking at nocnoc.com

Using JSON API server instead SQL and NoSQLDatabase

## install
```sh
cd front-end
npm install
```

## start JSON API  server
```sh
  cd back-end
  npx json-server --watch db.json --port 8000
  
```

## start 
```sh
  npm start
```
